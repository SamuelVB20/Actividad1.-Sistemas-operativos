1. ¿Qué es un Sistema Operativo?

Es un software fundamental que gestiona los recursos del hardware y proporciona servicios a los programas de aplicación, actuando como intermediario entre el usuario y el hardware.

2. Componentes de un Sistema Operativo

- Núcleo o Kernel
- Gestor de procesos
- Gestor de memoria
- Sistema de archivos
- Interfaz de usuario (gráfica o de línea de comandos)
- Controladores de dispositivos

3. Tareas del Sistema Operativo

- Administrar recursos del sistema
- Ejecutar aplicaciones
- Controlar dispositivos
- Manejar archivos
- Gestionar usuarios y procesos

4. Funciones de un Sistema Operativo

- Controlar el hardware
- Ejecutar y administrar programas
- Gestionar la memoria
- Proporcionar seguridad
- Manejar entrada/salida

5. Características de un Sistema Operativo

- Multitarea
- Multiusuario
- Portabilidad
- Seguridad
- Escalabilidad
- Estabilidad

6. Tipos de Sistema Operativo

- Monousuario
- Multiusuario
- Monotarea
- Multitarea
- En tiempo real
- Distribuido
- Móvil

7. Sistemas Operativos Para PC

- Windows
- macOS
- Linux (Ubuntu, Fedora, Debian, etc.)

8. Sistemas Operativos Móviles

- Android
- iOS
- HarmonyOS

9. Sistema Operativo según el Ordenador y el Tipo de Aplicaciones

- Para servidores: Linux Server, Windows Server
- Para PC: Windows, Linux, macOS
- Para dispositivos embebidos: FreeRTOS, VxWorks
- Para móviles: Android, iOS

10. Compatibilidad de los Sistemas Operativos

Se refiere a la capacidad del sistema para ejecutar aplicaciones, drivers y dispositivos correctamente, dependiendo de su arquitectura (32/64 bits), kernel y API.

11. ¿Cuál es la función básica de un Sistema Operativo?

Administrar los recursos del hardware y proporcionar una interfaz al usuario para interactuar con el sistema.

12. Mencione tres Sistemas Operativos más utilizados

- Microsoft Windows
- macOS
- Linux (especialmente Ubuntu)

13. Menciona algunas tareas que realizan los Sistemas Operativos

- Carga y ejecución de programas
- Gestión de memoria
- Administración de archivos
- Control de dispositivos periféricos
- Gestión de usuarios

14. ¿Qué es un sistema operativo multitareas?

Es aquel que permite ejecutar varios procesos al mismo tiempo, compartiendo recursos del sistema.

15. ¿Qué es un sistema operativo multiusuario?

Es aquel que permite que varios usuarios usen el sistema simultáneamente con sesiones independientes.

16. ¿Cómo funciona un sistema operativo?

Actúa como intermediario entre el hardware y las aplicaciones, gestionando procesos, memoria, archivos, entrada/salida y la interfaz con el usuario.

17. ¿Cómo se utiliza un sistema operativo en un entorno gráfico?

A través de interfaces gráficas (GUI) como ventanas, íconos y menús, usando dispositivos como mouse y teclado.

18. ¿Cuándo inicia el funcionamiento del Sistema Operativo?

Justo después del encendido del equipo, tras la ejecución del BIOS/UEFI.

19. Mencione las dos interfaces que forman parte del Sistema Operativo

- Interfaz de línea de comandos (CLI)
- Interfaz gráfica de usuario (GUI)

20. ¿Qué es un proceso en entorno de los sistemas operativos?

Es un programa en ejecución, incluyendo su estado, memoria, registros y recursos asignados.

21. ¿Qué es un sistema operativo distribuido y centralizado?

- Distribuido: coordina múltiples computadoras conectadas en red como si fueran una sola.
- Centralizado: todos los procesos y datos se gestionan en una única máquina.

22. Mencione los sistemas de archivos utilizados por los Sistemas Operativos

- FAT32, NTFS (Windows)
- ext4, XFS (Linux)
- APFS, HFS+ (macOS)

23. Mencione otras características actuales de los sistemas operativos

- Soporte para virtualización
- Administración de energía
- Actualizaciones automáticas
- Soporte para redes
- Seguridad avanzada con firewalls y cifrado

24. ¿Qué permite la exigencia de muchos controladores en un sistema operativo?

Permite la compatibilidad con una amplia variedad de dispositivos de hardware, asegurando su correcto funcionamiento.
